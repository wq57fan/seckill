## seckill秒杀系统
* Spring + Spring MVC + Mybatis 构成的秒杀功能
* 遵循 restful 接口
* 产品秒杀列表、秒杀功能等
* 进行高并发优化

## 开发环境
* Windows 10 x64
* IntelliJ IDEA 2018.1.5 x64
* MySQL 8.0.11
* Bootstrap
* Jquery
* redis
